# Installation

**If use TACKPAR770 put your helper bot Username Without @ in bot.lua, line 2

```sh
# Let's install the bot.
cd $HOME
git clone https://github.com/TACKPAR770/TACKPAR770-BDReborn.git
cd TACKPAR770
chmod +x beyond.sh
./beyond.sh install
./beyond.sh 
# Enter a phone number & confirmation code.

# For Auto Launch:
cd BDReborn
chmod 777 autobd.sh
screen ./autobd.sh
```
